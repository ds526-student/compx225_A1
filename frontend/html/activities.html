Activities

Python Programming Workshop - 18/07/25

Today we had a workshop on Python programming.

For those new to the language, we covered the basics of Python including syntax, data types, and control structures. For the more experienced, we explored pygame for game development as we learnt to create a mini top down shooter game. It was a great opportunity for everyone to learn and share knowledge about Python programming.

Guest Speaker - 25/07/25

Today we had a guest speaker, Jonathan Rogers, technical director at Grinding Gear Games located in Auckland. As one of the founders of the company, he shared his journey in the gaming industry and provided insights into game development. He discussed the challenges and successes he has faced throughout his career, and provided valuable advice for those interested in pursuing a career in game development.

It was an inspiring session that motivated many of us to continue pursuing our passion for programming and game development.
